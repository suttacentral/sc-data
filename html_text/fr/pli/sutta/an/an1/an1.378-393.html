• Aṅguttara Nikāya

Répétitions sur une acquisition

1.378

Vacana 378Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui séjourne en forêt.

1.379

Vacana 379Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui se nourrit d'aumônes.

1.380

Vacana 380Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui porte des haillons abandonnés comme déchets.

1.381

Vacana 381Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui porte seulement trois robes.

1.382

Vacana 382Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui fait des discours sur le Dhamma.

1.383

Vacana 383Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui connaît la Discipline par cœur.

1.384

Vacana 384Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un de très instruit.

1.385

Vacana 385Véritablement, bhikkhous, ceci est une acquisition: être un ancien.

1.386

Vacana 386Véritablement, bhikkhous, ceci est une acquisition: avoir bonne allure.

1.387

Vacana 387Non traduit.

1.388

Vacana 388Non traduit.

1.389

Vacana 389Véritablement, bhikkhous, ceci est une acquisition: être d'une bonne famille.

1.390

Vacana 390Véritablement, bhikkhous, ceci est une acquisition: avoir belle allure.

1.391

Vacana 391Véritablement, bhikkhous, ceci est une acquisition: être un bon orateur.

1.392

Vacana 392Véritablement, bhikkhous, ceci est une acquisition: avoir peu de désirs.

1.393

Vacana 393Véritablement, bhikkhous, ceci est une acquisition: être en bonne santé.
