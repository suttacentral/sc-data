Les confiances les plus éminentes

Nota Bene: Ceci est une ébauche. La relecture n'est pas terminée.

Bhikkhous, il y a ces quatre confiances les plus éminentes. Quelles sont ces quatre?

Bhikkhous, parmi les êtres, qu'ils soient sans pieds, à deux pieds, à quatre pattes ou à multiples pattes, matériels ou immatériels, percetifs, apercetifs ou ni-perceptifs-ni-aperceptifs, le Tathagata, l'arahant correctement et pleinement éveillé est déclaré être le plus éminent. Bhikkhous, ceux qui ont confiance dans le Bouddha ont confiance dans ce qui est le plus éminent. Et pour ceux qui ont confiance dans ce qui est le plus éminent, le résultat est le plus éminent.

Bhikkhous, parmi les phénomènes construits, la noble voie à huit composantes est déclarée être la plus éminente. Bhikkhous, ceux qui ont confiance dans la noble voie à huit composantes ont confiance dans ce qui est le plus éminent. Et pour ceux qui ont confiance dans ce qui est le plus éminent, le résultat est le plus éminent.

Bhikkhous, parmi les phénomènes, construits ou inconstruits, le détachement est déclaré être le plus éminent, c'est-à-dire la maîtrise la fierté, l'élimination de la soif, la suppression de l'attachement, l'arrêt du cycle [des renaissances], l'élimination complète de l'appétence, le détachement, la cessation, l'Extinction. Bhikkhous, ceux qui ont confiance dans le détachement ont confiance dans ce qui est le plus éminent. Et pour ceux qui ont confiance dans ce qui est le plus éminent, le résultat est le plus éminent.

Bhikkhous, parmi les communautés et les congrégations, la Communauté des disciples du Tathagata est déclarée être la plus éminente, c'est-à-dire les quatre paires d'individus, les huit [types de] personnes, cette Communauté des disciples du Fortuné qui est digne de dons, digne de soutien, digne d'offrandes, digne de salutations respectueuses, qui est un incomparable champ de mérite pour le monde. Bhikkhous, ceux qui ont confiance dans la Communauté ont confiance dans ce qui est le plus éminent. Et pour ceux qui ont confiance dans ce qui est le plus éminent, le résultat est le plus éminent.

Voici, bhikkhous, quelles sont ces quatre confiances les plus éminentes.
