56.129. Meninggal Dunia dari Alam Setan (4)

… “Demikian pula, para bhikkhu, hanya sedikit makhluk-makhluk yang, ketika meninggal dunia dari alam setan, terlahir kembali di antara para deva. Tetapi banyak sekali makhluk-makhluk yang, ketika meninggal dunia dari alam setan, terlahir kembali di alam neraka.”

56.130. Meninggal Dunia dari Alam Setan (5)

… “Demikian pula, para bhikkhu, hanya sedikit makhluk-makhluk yang, ketika meninggal dunia dari alam setan, terlahir kembali di antara para deva. Tetapi banyak sekali makhluk-makhluk yang, ketika meninggal dunia dari alam setan, terlahir kembali di alam binatang.”
